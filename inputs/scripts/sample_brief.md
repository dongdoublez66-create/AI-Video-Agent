# 示例剪辑需求

类型：日常 vlog / 产品宣传片

目标平台：抖音

成片时长：30-60 秒

风格：干净、节奏明确、有轻微高级感，默认淡入淡出。

剧本结构：

1. 开头 3 秒：用最有吸引力的镜头开场，字幕直接抛出亮点。
2. 中段：按故事或卖点组织素材，每 4-6 秒切一次。
3. 重点：产品细节、人物反应或环境变化处可以建议高级转场。
4. 结尾：总结一句话，并保留 1 秒品牌/行动引导。

字幕要求：

- 竖屏大字幕，白字深色阴影
- 关键卖点可以用黄色强调
- 不要遮挡人脸和产品主体

音乐要求：

- Agent 自动选择 BGM
- 人声优先，BGM 音量保持在 15%-25%
